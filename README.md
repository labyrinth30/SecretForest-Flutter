### 프로젝트 실행 방법
1. Flutter SDK 설치
2. Android Studio 또는 Visual Studio Code 설치
4. 이 레포지토리를 클론합니다.
5. 터미널에서 `flutter pub get` 명령어 실행
6. `lib/main.dart` 파일을 Chrome으로 실행합니다.
